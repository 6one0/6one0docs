        # 这是个什么？

        本篇作者：hellomadf (opens new window)

        欢迎来访！

        # 怎么来的？

        这是一个基于vuepress 1.x的静态文档网站，并开源于Github，部署于Github Pages上。本站是隶属于6one0下分的文档网站——6one0docs，由vuepress强力驱动搭建！

        # 干什么的？

        本站主要是关于6one0中所指向的帮助文档，你可以在此了解到许多东西，且完全免费！

        # 开发同款？

        你只需到Vuepress的官网即可学习搭建由vuepress驱动的静态网站，你也可以到本网站的开源地址直接下载源码压缩包进行套用（注意参考vuepress帮助文档中对vuepress所需运行环境的依赖的说明），祝你成功！
